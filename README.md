Calculator (calculator.js)

Features


add(num1, num2), subtract(num1, num2), multiply(num1, num2), divide(num1, num2) — perform the four basic arithmetic operations and return the result.
Every calculation is automatically logged to a history array as an object containing both operands, the operator used, and the result.
displayHistory() — prints the full calculation history to the console, or lets the user know if no calculations have been made yet.


Running It

bashnode calculator.js
